# Gerador de Senha

Aplicativo gerador de senha simples criado em React Native

## Instalação

Clone no repositório

```bash
git clone https://github.com/vespidhook/geradorDeSenha.git
```
Instale as dependências

```bash
yarn ou npm install
```
Inicie o projeto

```bash
yarn start
```
